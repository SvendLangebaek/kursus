# Sammenligninger der kan bruges i en *if* konstruktion

### Lig med: x == y
### Ikke lig med: x != y
### Større end: x > y
### Mindre end: x < y
### Større end eller lig med: x >= y
### Mindre end eller lig med: x <= y
### Objekt sammenligning: x is y
Objekt sammenligning ofte bruges sammen med None
### Medlem af: x in y
Bruges sammen med f.eks. list og string
## Logik
### Og: x and y
### Eller: x or y
### Ikke: not x



# Eksempler
### 1 == 3 : False
### 3 == 3.0 : True
### 3 == Decimal(3) : True
### 1 != 3 : True
### 1 > 3 : False
### 1 < 3 : True
### 2 >= 2 : True
### 2 <= 2 : True
### x = None<br/>x is None : True
### 1 == 1 and 1 != 1 : False
### 1 == 1 or 1 != 1 : True
### not 1 == 1 : False
