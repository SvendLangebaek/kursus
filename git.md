## Git er det værktøj vi bruger til at organisere vores arbejde i teams
## Hvis du ikke allerede har git installeret så download og installer git fra https://git-scm.com/
## Check lige i settings at pycharm har fundet git correct, ellers prøv genstart. Hvis den ikke finder git så sæt stien manuelt

## For at bruge git bruger man ofte (men ikke altid) et online repository.<br/> Man kunne f.eks. vælge github, gitlab eller gitea.<br/>Herfra antager vi at du har valgt github https://github.com/

## Opret en bruger på github
## Opret et repository vær opmærksom på om du opretter et "public" eller "private" repository

## Man kan vælge at bruge git udelukkende fra command line eller fra pycharm
## Man behøver ikke tilknytte sin github konto i pycharm for at benytte sig af git features i pycharm 

## Clone dit repository. Det betyder at du laver en lokal kopi. Senere kommer vi til at skubbe kode tilbage til github
### - Find stien til dit repository, grøn knap i højre side af github repository
### - Formen på stien er enten<br/> git@github.com:\<bruger navn\>/\<repository navn\>.git<br/>eller<br/>https://github.com/<bruger navn\>/\<repository navn\>.git
### - https er nemmere at sætte op, men kræver at du logger ind med password<br/>Ønsker du at sætte ssh op så er der en god guide her<br/> https://docs.github.com/en/authentication/connecting-to-github-with-ssh

## Inviter gerne andre kursister til dit repository :) Dette gøres i github repository under settings
